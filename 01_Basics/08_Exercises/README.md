# Python 3.11+ 综合练习（Exercises）

本目录把 `01_Basics/01_Comments` 到 `01_Basics/07_Data_Types` 的练习集中到一起：**每题一个文件**，直接运行即可（题目 + 参考答案 + 自测输出）。

## 怎么使用

在仓库根目录执行：

- 先看索引：`python3 01_Basics/08_Exercises/01_overview.py`
- 进入某一章练习索引（例）：`python3 01_Basics/08_Exercises/05_Input/01_overview.py`
- 运行某一道题（例）：`python3 01_Basics/08_Exercises/05_Input/03_fake_input_and_ask_until_valid.py`

每个练习文件开头 docstring 会写清楚“题目是什么”；文件里的实现就是“参考答案”；`main()` 会跑一组最小自测（[OK]/[FAIL]）。
