# Python 3.11+ Math and Statistics (Chapter 28)

This chapter covers numeric tools beyond basic operators:
- math module basics
- statistics module basics
- decimal context and rounding
- fractions advanced usage

---

## 1) How to run

From repo root:

- Index: `python3 01_Basics/28_Math_Statistics/01_overview.py`
- Single lesson: `python3 01_Basics/28_Math_Statistics/02_math_basics.py`
- Exercises index: `python3 01_Basics/28_Math_Statistics/Exercises/01_overview.py`

---

## 2) Key topics checklist

- math: sqrt, log, trig, isclose
- statistics: mean, median, pstdev
- decimal context and quantize
- fractions and limit_denominator
